## 실행 절차 (예: 메모장에 hello 입력)

1. 대상 창(빈 메모장 등)을 연다.
2. `capture_screen.py --mode full` 로 캡처 → 입력 필드 좌표 확인.
3. `computer_action.py click --x <X> --y <Y>` → 포커스.
4. `computer_action.py type --text "hello"`.
5. 재캡처로 `hello` 반영 확인.
